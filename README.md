## Project Description
This project implements a University Management System using a relational SQL database. It manages students, courses, instructors, modules, and enrollments.

## Database Tables
- Students
- Courses
- Instructors
- Modules
- Enrollments

## Files
- database/schema.sql → database structure
- database/insert_data.sql → sample data
- database/queries.sql → queries
- design/er_diagram.png → ER diagram

## How to Run
1. Run schema.sql
2. Run insert_data.sql
3. Run queries.sql
